最短路径算法:Dijiskstra算法

Aug 31, 2018 | Computer Network

路由选择算法中都要用到求最短路径算法，比较经典的有Bellman-Ford算法和Dijskstra算法，它们的思路不同但得到的结果一样。
寻找源结点到其它各结点的最短路径，每次都能找到一个到源结点的最短路径。

令D(v)为源结点到结点v的距离，它是源结点沿某一条路径到v的所有链路之和。
令l(I,j)为结点i到j的距离。
