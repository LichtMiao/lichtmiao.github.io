链表——反转单向和双向链表

Aug 29, 2018 | Data Structures and Algorithms

题目：反转单向链表，函数返回新链表的头节点。

实现：
方法一：最简单的就是使用栈，但复杂度为O(N)。

方法二：在原来链表上进行操作，使复杂度为O(1)。
next = head.next;
主要搞清楚指针（链接）和实际值。
