链表——将单向链表划分成左边小、中间相等、右边大的形式

Aug 29, 2018 | Data Structures and Algorithms

题目：将单向链表按某值划分成左边小、中间相等、右边大的形式（其中每部分的顺序与在原链表中的顺序相同）.

实现：

  1. 将原链表中的所有节点分到三个独立链表中，每个链表分别存储比k小的值，与k相等的值和比k大的值。
  2. 最后把这三个链表串联起来。
    注意对null节点的判断。
