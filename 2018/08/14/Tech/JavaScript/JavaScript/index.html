  3. 闭包（closure）
    闭包是能够读取其它函数内部变量的函数。
    在JavaScript中，闭包为定义在函数内部的函数，它可以访问局部变量。同时让那个这行变量的值始终保持在内存中。

  4. 闭包的使用注意
    1）由于闭包会使得函数中的变量都保存在内存中，内存消耗很大，因此不能滥用闭包。
    2）闭包会在父函数外部改变父函数内部变量的值。
