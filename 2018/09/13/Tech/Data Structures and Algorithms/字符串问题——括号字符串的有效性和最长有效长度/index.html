字符串问题——括号字符串的有效性和最长有效长度

Sep 13, 2018 | Data Structures and Algorithms

题目：给定一个字符串，判读是不是整体有效的括号字符串。
例：
str=”()”，返回true
str=”(())”，返回true
str=”())”，返回false
str=”()a()”，返回false
实现

  1. 遍历str，若不是括号，则直接返回false。
  2. 遍历到每个字符，检查括号的数量，如果’)’更多，则返回false。
  3. 遍历完，若’(‘的数量和’)’相等，则返回true，否则返回false。
