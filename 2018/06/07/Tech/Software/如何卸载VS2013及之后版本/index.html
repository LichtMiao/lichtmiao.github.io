如何卸载VS2013及之后版本

Jun 7, 2018 | Software

一. 问题

我之前安装了Visual Studio 2013 community with Update 4 （在线安装包已经被我删除，而官网只有VS2013 express with update 4版本的）。MS的性冷淡风格简直对我的胃口，于是我想能不能用VS作为IDE工具编码java。去网上搜了下还真有，但不能 debug。

如何用VS进行java编程的参考：

  • 如何用visual studio编译java源代码
  • Compiling Java in Visual Studio
  • JAVA TOOLS FOR VISUAL STUDIO

按照里面的步骤走的时候，想起当年安装的时候自定义，很多东西没装，没了安装包也无法修复，所以我打算卸载后重装。

此时问题出现了，VS根本就卸不掉，无论是用360安全卫士，还是直接在控制面板中卸载，根本就没反应。网上查发现VS卸载不掉是普遍问题，而且容易出现卸载不干净的问题。

二. 如何卸载VS

  1. 若还保留VS的安装包，直接打开“卸载”或“修复”。
  2. 若没有安装包，
    MS在github上提供了卸载方案，下载：VisulStudioUnistaller,
    以管理员身份运行Setup.ForcedUninstall.exe文件

    注意会卸载掉2013之后的VS。

三. 重装

VS已经更新到2017，为了防止软件出现不能解决的错误，这次我打算装Visual Studio 2013 community with update 5

四. 其它

  1. 微软的捆绑真是很麻烦了。虽然C盘中的残留应该删掉了，但控制面板中还是有显示，无法删除。算了不管了，以后发现删除的方法再更新。
  2. 用360安全卫士带的“系统盘瘦身”工具，转移虚拟内存、删除了系统自动备份区的文件、将个人文件转移到F:\360MoveData，时C盘空出了10G多的空间。还是挺有用的。
  3. 搜软件的时候发现了Visual Studio Code，它是一款编辑器，风格较类似Sublimetext3；而Visual Studio是编译器。
  4. 下载Markdownpad编辑器。
  5. 对VS除了风格特别喜欢其实没啥意思，，还是eclipse用着顺手。反正我编辑工具用的Notepad++，无论是VS还是eclipse都只是作为编译工具。
