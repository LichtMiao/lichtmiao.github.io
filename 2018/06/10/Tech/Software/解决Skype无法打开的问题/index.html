Skype

Jun 10, 2018 | Software

早上开会发现双击skype无响应，上周才使用过，应该不是电脑的问题，可能软件更新过出现了些问题。（先关闭了360，防火墙啥的，防止杀毒软件误杀），按网上的教程：

  1. 在开始 -> 运行栏中输入“%appdata%\skype”,回车找到Skype 的配置文件存储目录；
  2. 删除目录中的”shared.xml、shared.lck”两个文件
  3. 成功删除文件后，重新启动Skype 即可。

解决：

  1. 重新上官网下载桌面版，安装后按照网上教程进行解决
    但是我在第一步就卡住了，根本找不到skype配置文件的根目录。猜测是没安装成功或是版本安装错误。后面发现确实是版本的问题，直接上英文的官网下载的就是有问题。
  2. 重新搜到Skype简体中文版官网
    安装过程与英文版面有些不同，但最终能够使用。看来是版本的问题。
  3. 还有Sype网页版，具体功能应该跟桌面版一样，没有测试过。

其它：

Skype被Microsoft收购，搜的时候发现一个工作版：skype for business，安装后会自动添加到Microsoft office里面，但登录的时候发现不能登自己的邮箱账号，需要输入地址？？？反正就是不能用自己的账号，所以不要下载这个版本的。
